# Qué es efac.xyz

Nos gustaría cambiar la enseñanza en este país. Una iniciativa pública frente a las academias privadas.

  - Pequeños grupos
  - Ayuda mutua
  - Correcciones de ejercicios

> La academia pública, una filosofía de estudio
